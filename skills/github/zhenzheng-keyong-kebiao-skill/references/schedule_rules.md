# 课表规则

## 教学进度课表

输入：

- 教学周数
- 周课时
- 总课时
- 单元及建议课时
- 复习周、考试周、节假日
- 匹配到的课程标准

流程：

1. 扣除复习周、考试周、节假日。
2. 按单元建议课时分配可用周次。
3. 每个单元附 1-3 个主要标准 code。
4. 生成活动建议和评价证据。
5. 检查总课时、保留周、空周和低置信度匹配。

## 日常节次周课表

硬约束：

- 每学科周课时数量准确。
- 每天节次数不超过上限。
- 固定活动不能被覆盖。
- 教师不可用时间不能排。
- 同一教师不能同时教两个班。
- 专用教室不能冲突。

软约束：

- 语数英尽量排上午。
- 体育、艺术、劳动不要全部集中一天。
- 同一学科不要连续太多节。
- 科学、劳动、信息科技尽量安排连续或靠近的实践时段。
- 每天负荷均衡。

MVP 可用贪心分配 + 冲突检查；正式版再接约束求解。
