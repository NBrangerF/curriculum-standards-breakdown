# 示例

## 示例 1：查课标

用户：

```text
给我三年级科学植物相关课程标准
```

处理：

1. 识别三年级，按当前数据口径推断 H2，并标注需确认。
2. 学科为 `science`。
3. 关键词为“植物”。
4. 检索真实标准，按领域分组。

## 示例 2：上传计划生成进度表

用户上传教学计划后：

1. 解析 plan JSON。
2. 校验年级、学科、教学周数、单元课时。
3. 匹配标准。
4. 生成教学进度表。
5. 输出覆盖分析和人工确认项。

## 示例 3：日常周课表

用户：

```text
三年级每周语文 6、数学 5、英语 3、科学 2、体育 3、艺术 2、劳动 1、道法 2，帮我排一张周课表。
```

处理：

1. 将每周课时转换为 subject-period map。
2. 根据每天节数生成空表。
3. 先排硬约束，再优化软约束。
4. 输出约束检查。
