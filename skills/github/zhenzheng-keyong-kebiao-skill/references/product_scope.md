# 产品范围

## 产品目标

“真正”能用的课标 Skill 不是网页入口，而是教学规划助手。它要把课程标准、教学计划和课表生成连接起来：

- 按学科、年级、学段、领域、关键词查询真实课程标准。
- 将学校课程方案或教师教学计划匹配到真实课程标准。
- 生成教学进度课表和日常周课表。
- 输出覆盖分析，指出遗漏、重复和低置信度匹配。

## MVP

1. 按学段/学科查课标。
2. 解析结构化 `.json` / `.md` / 清晰表格型计划。
3. 匹配课程方案单元到标准。
4. 生成 Markdown 教学进度表。
5. 输出覆盖分析。

## 暂不承诺

- 不保证覆盖所有义务教育学科，除非数据源已补齐。
- 不把教材版本映射当作已有能力。
- 不处理扫描图片，除非明确接入 OCR。
- 不保证复杂多班级多教师排课最优，只先检查硬约束。
