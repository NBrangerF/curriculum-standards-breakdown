# 学习脉络内容与文案契约

## 用户可见名称

- 产品入口：学习脉络
- 副标题：先掌握什么 · 接下来解锁什么
- 三个方向：需要先掌握、当前知识点、将会解锁
- `progression` Lens：学段进阶

## 空状态

| 条件 | 文案 |
|---|---|
| 入边未审核 | 当前尚无经证实的先修关系。 |
| 入边已审核且为空 | 这是当前已审核学习范围内的起点。 |
| 出边未审核 | 当前尚无经证实的后续解锁。 |
| 出边已审核且为空 | 这是当前已审核学习范围内的终点。 |
| 仅存在学段进阶 | 暂无经证实的认知先修；以下展示经核验的学段进阶。 |

## 禁止表达

- 不称课程标准、领域或能力标签为“知识点”。
- 不称 `progression` 为先修、前置条件或解锁。
- 不将 taxonomy 父子关系、详情相邻导航或机器候选边显示为确认的 prerequisite。
- 不在公开页面展示内部审核、年级归属或机器置信度详情。
